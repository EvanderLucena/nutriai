## Major Decisions

- **Data layer needs real backend** — Mock data in `data.jsx` limits tudo. Sem persistência, sem multi-usuário.
- **WhatsApp integration is the core moat** — A extração por IA via WhatsApp é o diferencial competitivo central. Vale investir pesado.
- **Frontend está em protótipo produtivo** — Single HTML com React CDN+Babel. Funciona como demo/prototype, mas precisa evoluir para app real.

## Project Trajectory

Prototype → MVP → Product → Platform
